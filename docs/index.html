Installation

1

Add Marketplace

claude plugin marketplace add brolag/neural-claude-code-plugin
2

Install Plugin

claude plugin install neural-claude-code@brolag --scope user
3

Set Up Hooks

export CLAUDE_PLUGIN_ROOT="$HOME/Sites/neural-claude-code-plugin" bash "$CLAUDE_PLUGIN_ROOT/scripts/setup-hooks.sh"
4

Initialize Project

cd /path/to/project && claude > setup claude
